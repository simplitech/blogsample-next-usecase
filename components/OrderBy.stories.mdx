# OrderBy

## Primary
OrderBys are components that shows the sorting controller and has callbacks to ask for the desired sorting.

<Story name="Primary"
       args={{
         fieldName: "id",
         children: "ID",
         controller: {
           orderBy: undefined,
           sortOrder: undefined,
           setOrderBy: (column) => action('setOrderBy')(column)
         },
         parentWidth: 'lg'
       }}>
  {StoryComponent.bind({})}
</Story>

## Order by ID ASC
The component is not responsible for the real sorting, that is why it has the setOrderBy callback.
